Hamiltonian Paths and the Icosian Game

In graph theory, a Hamiltonian path is a path on which each node is visited only once. In a Hamiltonian cycle it is possible to jump from the last node of the path to the first node of the path, thus forming a cycle. Hamilton tried to market the so-called Icosian Game in 1856, where players should find a Hamiltonian path/cycle in the dodecahedron graph. Below you can find a single-player version of this game.
